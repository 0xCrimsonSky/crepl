# CREPL

A minimal REPL (Read-Eval-Print Loop) built in C for learning and experimentation.

The program reads input from the terminal, processes it, prints the result, and continues running until you exit.

## Build

```bash
make
```

## Run

```bash
./app
```

## Clean

```bash
make clean
```

## Project Files

- `main.c` — contains the REPL loop
- `Makefile` — build instructions

## Purpose

This project is a small exercise to understand the basics of:

- C programming
- terminal input handling
- simple parsing and evaluation
- REPL-style program flow
